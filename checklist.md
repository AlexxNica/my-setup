- [ ] Update to the latest OSX patch
- [ ] Set new system password
- [ ] Turn on hard drive encryption
- [ ] Trackpad
	- [ ] Turn on 1 click touch

- [ ] Spotlight: change to `alt A` for Spotlight search
- [ ] Global: `cmd shift +` for Window > Zoom
- [ ] Dock
	- [ ] Move to left side
